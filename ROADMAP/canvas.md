# Canvas — Visual Project Board

> **Status:** Planning — 2026-03-25

Canvas is a self-hosted visual project manager. A freeform canvas where notes, tasks, images, links, and checklists can be placed and arranged freely. Built for planning and tracking creative projects, ideas, and anything else that doesn't fit neatly into a list.

**Stack:** SvelteKit · Fastify · SQLite
